
# MoveOnSuiBootcampFinalProject

## Getting Started

1. Run the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   

2. Open [http://localhost:3000](http://localhost:3000) in your browser.

3. Edit `pages/index.js` to modify the page (it auto-updates).

## API Routes

- Access API routes at [http://localhost:3000/api/hello](http://localhost:3000/api/hello).
- Edit the endpoint in `pages/api/hello.js`.
- The `pages/api` directory maps to `/api/*` for API routes.
